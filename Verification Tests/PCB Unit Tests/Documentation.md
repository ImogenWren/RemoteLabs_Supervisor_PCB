## Test 2: Check Voltages:
Connect power supply to the PCB and measure the voltages at the following points. Ensure not to connect any valuable components.
- J31 - 5V Pin
- J30 - 12V Pin
- A3 - VBAT
- J6 - 5V Pin
- J34 - 5V Bus
- J33 - 12V Bus
- J35 - 5V Bus
- J36 - 3.3V Bus
- A4 - Pin 16
